# analytics_engineering_journey

Портфолио аналитика данных → Analytics Engineer → AI‑enhanced

---

## 👋 О себе

Аналитик данных с сильным SQL и продуктовым мышлением.  
  
Прохожу путь: **Data Analyst → Data Engineer → AI‑enhanced Analyst**.

**Ключевые навыки:**
- SQL (оконные функции, оптимизация запросов, EXPLAIN)
- Python (pandas, ETL, Jupyter)
- Power BI (дашборды, DAX)
- Airflow, Docker, PostgreSQL
- AI‑инструменты (Cursor, GPT, Telegram‑боты)

## 🚀 Ключевые проекты

### 1. Анализ оттока клиентов (SQL + Power BI)
- Оконные функции, когорты, retention
- Дашборд с фильтрами и DAX‑мерами
- [Скриншоты](03_bi/power_bi/screenshots/)

### 2. ETL‑пайплайн на Python + Airflow
- Загрузка, очистка, витрина данных
- DAG с зависимостями и расписанием
- [Код ETL](04_etl/), [DAG](05_airflow/dags/)

### 3. Docker‑стек для локальной разработки
- PostgreSQL + pgAdmin + Airflow
- Одна команда `docker‑compose up`
- [Инструкция](06_docker/README.md)

### 4. Telegram‑бот с GPT для аналитики
- Команды: /churn, /explain, /dashboard
- Интеграция с OpenAI API
- [Код бота](07_ai/telegram_bot_gpt.py)
